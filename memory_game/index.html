Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso or simple Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

Intrigued?

You should be! Not only is the game fun to play - especially for you competative folks out there - but scientists have discovered this type of game helps improve brain function. Not only do you need your memory to play, but it improves it as well!
